# Talha K. — Professional Portfolio

A stunning, animation-rich portfolio website inspired by modern developer portfolios. Built with Next.js 15, Framer Motion, GSAP, and Lenis smooth scrolling.

## Features

- **Extreme animations** — GSAP text reveals, Framer Motion scroll triggers, magnetic buttons, particle network background
- **Smooth scrolling** — Lenis-powered buttery scroll experience
- **Fully responsive** — Mobile-first design with animated mobile menu
- **Easy customization** — All content in `src/data/portfolio.ts`
- **Sections** — Hero, About, Projects, Skills, Services, Contact

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Customize Your Portfolio

Edit `src/data/portfolio.ts` to update:

- Personal info (name, email, phone, social links)
- About bio and stats
- Projects (title, description, tags, links)
- Skills and services

## Deploy

Deploy to [Netlify](https://netlify.com) or [Vercel](https://vercel.com):

```bash
npm run build
```

## Tech Stack

- Next.js 15 + React 19
